Usage

The script is used to to shorten URLs and provide 2 pages of ads to earn money , it has a powerful admin panel with statistics

This script can be used as stand alone to shorten URLs , or you can plug that script in your web application

that's why the template structure is as simple as possible to help you easily modify it.

1. Shorten single URL

  1. Go to the home page
  2. Enter the Link and other details
  3. Click send button
  4. you will get the shorten link

 

2. Multi URLs shorten

  1. Click multi
  2. paste the URLS
  3. URLs must be one per line
  4. click send button
  5. Copy the shorten URLs

Single URL shorten

This is the main functionality of the script it allows your visitors to paste a single URL to shorten and it has 4 options:

  1. LINK ( this field is required)

  2. NAME ( The name of the URL this field is optional)

  3. SIZE ( The size of the file if it's available this field is optional)

  4. OWNER ( the owner of the file this field is optional)

Multi URLs shorten

This option allows the users to shorten more than 1 URL at once, just paste the URLs one URL per line
